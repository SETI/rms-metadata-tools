===============
Developer Guide
===============

This guide is for developers who modify, extend, build, test, or release
``rms-metadata-tools``. It explains how the code is organized, how the pieces
cooperate, and how to add support for new collections and new geometry columns.
For installing and running the tool, see the :doc:`/user_guide/user_guide`.
