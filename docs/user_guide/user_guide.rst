==========
User Guide
==========

This guide explains how to install ``rms-metadata-tools`` and use it to generate
PDS3 index, geometry, and cumulative metadata tables (and their labels) for a
planetary science data collection. It is written for someone fluent in Python
and the command line who has not read the source.
